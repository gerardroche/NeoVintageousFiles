# NeoVintageousTree

A Sublime Text [NeoVintageous](https://github.com/NeoVintageous/NeoVintageous) plugin.

Key | Description
:-- | :----------
<kbd>a</kbd> | Add file.
<kbd>A</kbd> | Add folder.
<kbd>d</kbd> | Duplicate.
<kbd>f</kbd> | Find.
<kbd>m</kbd> | Move.
<kbd>f2</kbd> | Move.
<kbd>i</kbd> | Open in split.
<kbd>s</kbd> | Open in vertical split.
<kbd>t</kbd> | Open in tab.
